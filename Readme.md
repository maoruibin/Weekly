# Weekly 
打算工作日的每一天都去围绕一些看到的或者经历的事情，去写一点感悟、思考，然后每周末整理出来。

通过这种方式强制自己做一些稍微深入的思考和记录，同时与朋友更好的分享交流。

## 实践方式
每日晚上会发表短文在这个仓库的 issue 中，每一周为一个 issue, 每一天的短文会记录在下面的评论中，周末进行整理发布。
